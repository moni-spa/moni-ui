# Snackbar

Snackbars show short updates about app processes at the bottom of the screen

![Diagram of snackbar indicating the four parts of its anatomy](../../assets/snackbar/specs/01-diagram-of-snackbar-indicating-the-four-parts-of-its-anatomy-01148741.png)

1. Container
2. Icon (optional close affordance)
3. Action (optional)
4. Supporting text
